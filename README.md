# Americans for Propriety

Americans for Propriety is a nonpartisan civic drafting and education project
focused on institutional propriety, clear jurisdictional boundaries, and
responsible public policy engagement.

The site provides neutral policy background materials and letter-drafting tools
designed to help constituents engage thoughtfully and respectfully with public
officials at the state and federal level.

---

## Purpose

Public policy debates are often driven by urgency, partisanship, or incomplete
context. This project exists to support a different mode of civic engagement:
one grounded in clarity, restraint, and respect for institutional roles.

Americans for Propriety focuses on **how** government authority is exercised,
rather than advocating for particular political outcomes.

---

## What This Project Is

- A static, publicly auditable website
- A collection of issue-specific policy context pages
- Neutral, reusable letter templates for constituent outreach
- A tool for thoughtful civic participation

---

## What This Project Is Not

- Not an advocacy organization
- Not a political action committee (PAC)
- Not a fundraising platform
- Not a mass-emailing or automated outreach system
- Not a data collection or tracking service

This project intentionally avoids automated contact with public officials,
user accounts, analytics, or storage of personal information.

---

## Guardrails & Principles

Americans for Propriety is guided by the following principles:

- Respect for constitutional and statutory roles
- Clear boundaries between federal, state, and local authority
- Nonpartisan, fact-based policy discussion
- Transparency in tooling and methodology
- Restraint in civic engagement
- User control over all generated content

All letter templates are generated locally in the browser. Users are responsible
for reviewing, customizing, and sending any correspondence.

---

## Technology

- Static HTML, CSS, and JavaScript
- Hosted via GitHub Pages
- No backend services
- No cookies or tracking
- No third-party analytics

The repository is intentionally simple and readable.
